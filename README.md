# 🧠 F-Mind — Gestion du temps & finances

> **“60 minutes, c’est trop long. F-Mind te fait gagner du temps ET de l’argent — en 50 min.”**

---

## 🚀 À quoi ça sert ?

F-Mind est une **app web simple** pour :
- 🕒 Gérer ton temps avec un **minuteur 50 min** (inspiré du Pomodoro)
- 💰 Suivre tes finances (dépenses, revenus, objectifs)
- 📅 Organiser tes tâches quotidiennes
- 📈 Suivre ta productivité — sans complexité

> ✅ **Pas de compte, pas de pub, pas de tracking** — juste toi, ton temps, et tes objectifs.

---
